Development
Patrick August 5, 2025, 10:22am 1

Unplugging external drive should trigger a shutdown → this will be implemented soon: Removal of boot device will result in emergency shutdown. This will become available in Kicksecure 18 most likely.

As an additional feature, at the same time, we can provide an emergency key combination. Once pressed, a force shutdown sequence can be triggered.

We’re wondering about which key combination to use for that?

Ctrl+Alt+Del (sometimes called the “three-finger salute”): May not be the most suitable. Different plans for this: Ctrl+Alt+Del (three-finger salute) action

Potential choices:

  • Ctrl+Shift+Esc
  • Ctrl+Alt+Esc
  • Ctrl+Alt+Backspace

Ctrl+*+Esc combos slightly scare me since Ctrl+Esc is the shortcut for bringing up the task manager under KDE, and I believe Ctrl+Shift+Esc does so on Windows.

Another possible shortcut is Ctrl+Alt+End, which is used by Windows Remote Desktop to send a Ctrl+Alt+Delete keypress to a remote PC. Using the End key would be rather fitting for this also, since an emergency shutdown will immediately end whatever you were doing on the computer.
