Patrick:

Desirable, yes.

Not implemented because, quote Installation of Kicksecure on a USB

Why not just flash a persistent Kicksecure image to USB? - Such an image has not been developed yet due to complex technical challenges. Note that due to these complex challenges, almost no other Linux distributions offer such functionality either. To circumvent this, a “detour” through flashing a Kicksecure ISO is required to run the Kicksecure installer.

xianhua:

Is it possible to distromorph a raspberry pi (raspbian, Pi OS) into kicksecure?

Morphing Raspberry Pi OS into Kicksecure is untested and not recommended, RPiOS is a Debian derivative that departs from Debian in some substantial ways, morphing it into Kicksecure may cause issues.

You may be able to morph Debian 12 into Kicksecure on the Raspberry Pi 4B with decent results if you have sufficient technical skill. The boot process on the RPi is different enough from a traditional PC that it will cause some difficulties (for instance, hardening kernel parameters used by Kicksecure won’t be applied by default), and this isn’t explicitly tested or supported yet. It would be best to try following the existing Kicksecure distribution morphing documentation to start. This is not something we can support yet, so if you run into issues, you may be more-or-less on your own.

Official Raspberry Pi support is planned but not yet present. There isn’t an ETA on it.
