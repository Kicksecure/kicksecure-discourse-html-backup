Tengu:

So my question is whats the difference between grub live and a custom grub entry with toram noswap nohibernate flags?

grub-live mode mounts the root filesystem read-only, and uses one or more writable tmpfs filesystems (which exist purely in RAM) to store file changes. On the other hand, toram copies an ISO live filesystem into RAM entirely, so that you don‚Äôt need the boot media plugged in anymore for the system to remain running.

(Note that toram appears to be specific to live-tools and initramfs-tools. Kicksecure uses Dracut for initramfs management; the corresponding feature in Dracut appears to be rd.live.ram=1. I wouldn‚Äôt expect this to work with an installed Kicksecure system, but it might work on the ISO. Untested.)
