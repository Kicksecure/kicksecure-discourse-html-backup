sam August 28, 2024, 4:02pm 2

Hello. If you can’t install Kicksecure completely, then try installing grub-live from kicksecure github. I was able to do it - MX Linux got live mode.

And you can try uninstalling plymouth (but first, take a snapshot in Timeshift) - I’ve uninstalled it on many linux and after that installing ‘sudo apt install --no-install-recommends kicksecure-xfce-host’ was successful.

But first, wait for developer’s response
