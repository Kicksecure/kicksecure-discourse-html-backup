Point Release

What is a Point Release?

Major Changes

  • default package changes (Thanks to @nurmagoz!)
    • install more packages by default
      • hunspell-en-us
      • gpa
      • ffmpeg
      • pinentry-qt (or pinentry-x11)
      • pdf
      • youtube-dl
      • install vlc media player by default
    • no longer install wget by default
    • (Add/Remove Extra Packages - #4 by nurmagoz )
  • added kicksecure-welcome-page (Thanks to Hans!)
  • kicksecure-welcome-page: add 2022 end of year donation campaign banner
  • repository-dist man page: document --transport plain-tls, plain-tls-tor, onion or onion-tls
  • fixed Debian installation instructions
  • updated sdwdate onion sources (Thanks to @TNT_BOM_BOM!)
  • Work towards Change default shell from bash to zsh by default? - Development - Whonix Forum (Thanks to @nyxnor!)
  • improved build script --connection onion support
  • KVM: remove dependency on xserver-xorg-video-qxl [Help Welcome] KVM Development - staying the course - #501 by HulaHoop - KVM - Whonix Forum
  • fixed rare minor issue security-misc pam-info broken pipe bug - Support - Whonix Forum
  • pam-info (notify about how many remaining login attempts): now also working in xscreensaver

Upgrade

Alternatively, in-place release upgrade is possible upgrade using Kicksecure repository.

This release would not have been possible without the numerous supporters of Kicksecure!
