ike:

No, not without development effort.

Related, the long term Kicksecure development roadmap.

The long term concept:

Kicksecure – 6 Nov 23

Multiple Boot Modes for Better Security: an Implementation of Untrusted Root

Persistent User / Live user / Persistent Secureadmin / Persistent Superadmin / Persistent Recovery Mode

Whonix Forum – 6 May 20

multiple boot modes for better security: persistent user | live user |...

Development
status_open_issue_todo component_security

One thing hasn’t been considered yet at all: server support. grub boot menu isn’t easily accessible for many/most servers. How would these various boot modes be available for servers? Live mode: that probably makes little sense on servers. ...

sudoless is not coming anytime soon. Might be years if not decades. Unless there are volunteer contributors.
