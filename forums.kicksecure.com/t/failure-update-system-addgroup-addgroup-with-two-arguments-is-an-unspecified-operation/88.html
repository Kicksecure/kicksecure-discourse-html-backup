Debian testing isn’t supported at time of writing by Kicksecure. Install Kicksecure ™ inside Debian does not mention it. Should that change, that wiki page would be updated.

Seems pretty simple to fix. I’ll probably fix this soonish. Will post an update here once that’s done. However, there might be other incompatibles when attempting on install Kicksecure on top of Debian testing as long as it’s unsupported.
