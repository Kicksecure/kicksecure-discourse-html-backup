add the Mullvad repo

Optional trick for folks wanting less typing: If you install Mullvad Browser using the browser choice GUI (System Maintenance Panel > Install a Browser), it enables the Mullvad repo during the process. Coincidentally, that’s the same repo containing mullvad-vpn, so you can just install that package. (With the “Manage Software” button, the entire process can be done without typing a single command.) Afterwards, you can remove the browser if you want.

This comment is not an endorsement of Mullvad, their browser, or their VPN. It is an endorsement of the handy Kicksecure panel, though. The panel really simplifies a lot of common tasks for folks not familiar with shell. Even for shell experts, it still saves a lot of typing and offsets some productivity loss from having to reboot into sysmaint.
