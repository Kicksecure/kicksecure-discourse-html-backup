This might be a tricky one. Apple keyboard backlights are apparently controlled by a sysfs file, which likely requires administrator permissions to modify. Existing tools might not work with user-sysmaint-split installed unless they install a privileged background process that actually does the job of changing the backlight brightness. If you remove user-sysmaint-split though, you might be able to use something like brightnessctl to adjust the keyboard backlight.

I unfortunately do not have an Intel-based Macbook to test with, so I can‚Äôt really create a tool for adjusting this that integrates with Kicksecure. (I made a tool for adjusting the display backlight, which is part of Kicksecure 18; a similar tool would probably work for the keyboard backlight, but would require someone with compatible hardware to create the tool or add support for keyboard backlight changing to the existing tool.)
