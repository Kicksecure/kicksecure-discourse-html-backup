nearkick October 31, 2025, 5:01pm 4

I successed in interfacing the phone to the OS, and thunar is now showing the phone automatically.

Many sites were wrong with the packages.

I finally found the good combination, combining various sites infos :

  1. sudo apt install gvfs-backends gfvs-fuse
  2. disconnect from the kicksecure session, then reconnect to it so that changes are taken in account
  3. plug the phone
  4. on the phone select “file transfer”
  5. it’s done (for me) : the phone appears in thunar and is now browsable
