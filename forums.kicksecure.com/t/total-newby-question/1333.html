Patrick October 30, 2025, 10:45am 8

Scratch that. No screenshot needed.

When you boot an ISO it is expected that there’s only live mode.

If you want persistent mode, you need to boot, install using Calamares to (virtual or real) hard drive, power off, remove ISO, reboot.

Full instructions here:

Kicksecure Installation Guide. This guide will help you download the ISO image, create your bootable media, and install Kicksecure on your computer.
